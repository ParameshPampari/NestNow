\# Food Delivery Application



\## Overview



A full-stack food delivery platform that enables users to browse restaurants, explore menus, manage carts, place orders, and securely access the platform through JWT-based authentication.



\## Features



\* User Registration and Login

\* JWT Authentication and Authorization

\* Restaurant Management

\* Food Menu Management

\* Cart Management

\* Order Placement

\* Order History

\* Responsive User Interface

\* API Documentation with Swagger

\* Dockerized Deployment
